# Webpack+Es6+React
### 如何执行
####  1、将项目克隆到本地，cd到Webpack+Es6+React
```javascript
git clone git@github.com:kiss19861127/React.git
cd Webpack+Es6+React
```
#### 2、安装依赖
```javascript
npm install
```
#### 3、打包并执行
```javascript
npm run dev
```
#### 4、打开浏览器浏览 http://localhost:8100/
